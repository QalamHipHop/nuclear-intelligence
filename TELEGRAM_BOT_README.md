# XT.com Market Monitor Bot

این ربات برای نظارت هوشمند بر بازار صرافی XT.com طراحی شده است.

## قابلیت‌ها:
- **شناسایی اسپردهای شدید:** شناسایی جفت ارزهایی که اختلاف قیمت خرید و فروش آن‌ها بیش از ۰.۵٪ است.
- **شناسایی توکن‌های با نقدینگی کم:** بررسی عمق اوردر بوک و شناسایی توکن‌هایی که تعداد سفارشات کمی دارند.
- **شناسایی توکن‌های بدون اوردر بوک:** یافتن توکن‌هایی که عملاً هیچ سفارشی در آن‌ها ثبت نشده است.
- **اختلاف قیمت شدید:** مقایسه قیمت جفت ارزهای با پایه USDQ نسبت به USDT برای یافتن آربیتراژ.
- **توکن‌های مخفی:** شناسایی توکن‌هایی که در لیست ONLINE هستند اما در API عمومی فعال نیستند.
- **گزارش‌دهی خودکار:** ارسال گزارش هر یک ساعت یکبار به تلگرام.
- **لینک مستقیم:** تمام توکن‌ها دارای لینک مستقیم به صفحه معامله در صرافی XT هستند.

## نحوه اجرا:
1. ابتدا مطمئن شوید که کتابخانه‌های لازم نصب شده‌اند:
   ```bash
   pip install python-telegram-bot requests python-dotenv apscheduler
   ```
2. فایل `.env` را با توکن خود تنظیم کنید.
3. ربات را اجرا کنید:
   ```bash
   python telegram_bot.py
   ```
4. در تلگرام به ربات پیام دهید و دستور `/start` را بزنید.
5. برای بررسی دستی در هر لحظه، از دستور `/check` استفاده کنید.
